Rules:

Here you can find basic rules and hints about how to add your documentation here.

  • 1. Clone our repository

  • 1. Make new .html file with your documentation and format it by using our css file

  • 1. Create pull request

There is no need to contact anybody, just prepare locally your version of rusty-pipe repository and open pull request. In case of a need for discussion, this could be done at comments section at github's web page. Once we are ready to add your project, merging of commits will be managed by oldest members of our rusty-pipe group.
